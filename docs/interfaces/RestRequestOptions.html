Optional customBaseURL

customBaseURL?: string
  • Defined in src/rest/RestOptions.ts:55

A custom base URL to make requests to. Useful for making requests through, for example, a proxy / multi-application ratelimiter. Example: https://api.example.com/discord Make sure this URL does not end with a /. Note that the rest manager will also omit the API version from the route (.../v10/...).
